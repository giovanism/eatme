# EatMe Server

WebSocket server for [EatMe][home-eatme].

## Installation

Make sure you have [Maven][home-maven] installed.

* Run directly with Spring Boot:

  ```bash
  mvn clean spring-boot:run
  ```

* Pack and run:

  ```bash
  mvn clean package
  java -jar ./target/eatme-server-1.0.jar
  ```
